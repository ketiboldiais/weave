# Quadrilaterals
Quadrilaterals are represented by `Quad` nodes.

<Quad1 />


## Properties

### `W` **number**

The quadrilateral’s width.

### `H` **number**

The quadrilateral’s height.

### `Rx` **number**

The horizontal corner radius.

### `Ry` **number**

The vertical corner radius.

### `origin` **vector**

The quadrilateral’s origin. Visually, the origin is always at the top-left corner.
