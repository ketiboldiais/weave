# Quadrilaterals
Quadrilaterals are represented by `Quad` nodes.

<figure>
<Quad1 />
<Codeblock>
~~~tsx
import {
  axis,
  color,
  plane,
  quad
} from "@weave/loom";
import { Figure } from "./figure.tsx";

const rects = plane([
  axis("x"),
  axis("y"),
  [
    quad(1, 1).at(0, 1),
    quad(1, 2).at(1, 2),
    quad(1, 4).at(2, 4),
    quad(1, 1).at(3, 1),
    quad(1, 2.5).at(4, 2.5),
  ].map((q) => q.fill(color("beige")).stroke(color("goldenrod"))),
]).margin(50, 100).size(500, 300).x(0, 5).y(0, 5).figure();

const Quad1 = () => <Figure of={rects} />;
~~~
</Codeblock>
</figure>


## Properties

### `W` **number**

The quadrilateral’s width.

### `H` **number**

The quadrilateral’s height.

### `Rx` **number**

The horizontal corner radius.

### `Ry` **number**

The vertical corner radius.

### `origin` **vector**

The quadrilateral’s origin. Visually, the origin is always at the top-left corner.

## Methods

### `height`

Sets the quadrilateral’s [height](#h-number).

| Parameters | Returns |
| ---------- | ------- |
| _number_   | `Quad`  |

### `width`

Sets the quadrilateral’s [width](#w-number).

| Parameters | Returns |
| ---------- | ------- |
| _number_   | `Quad`  |

### `ry`

Sets the quadrilateral’s [vertical corner radius](#ry-number).

| Parameters | Returns |
| ---------- | ------- |
| _number_   | `Quad`  |

### `rx`

Sets the quadrilateral’s [horizontal corner radius](#rx-number).

| Parameters | Returns |
| ---------- | ------- |
| _number_   | `Quad`  |

### `at`

Sets the quadrilateral’s [origin](#origin-vector).

| Parameters                              | Returns |
| --------------------------------------- | ------- |
| (x: _number_, y: _number_, z: _number_) | `Quad`  |
